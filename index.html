Projeto Crescer 2015/1

Repositório utilizado pela CWI Software durante as aulas do projeto Crescer. Não conhece? Então clique aqui!

Configuração do ambiente de desenvolvimento

  1. Criando uma conta no GitHub
  2. Instalando a Java 8 (JDK)
  3. Instalando o BlueJ
  4. Instalando o Eclipse
  5. Instalando o SourceTree

Criando uma conta no GitHub

  1. Acesse o site github.com
  2. Na tela inicial será possível criar uma conta no site.
    1. Escolha um nome de usuário, email e senha
    2. (Dica) Evite apelidos no nome de usuário para facilitar a identificação! Prefira utilizar o seu nome/sobrenome (exemplos: marlonbernardes, bernardobrezende)
    3. (Dica 2) É possível alterar o nome de usuário após o cadastro.
    4. Clique no botão verde (sign up) para realizar o cadastro.

Instalando o Java 8 (JDK)

  1. Acesse a página de downloads da Oracle clicando aqui
  2. Em Java SE Development Kit 8u clique em Accept License Agreement
  3. Baixe a versão compatível com o seu sistema operacional
  4. Execute o arquivo e siga os passos de instalação

  5. Para verificar se a instalação deu certo, abra o prompt de comando/terminal e veja se o comando abaixo executa sem erros

    java -version

Instalando o BlueJ

  1. Acesse BlueJ.org
  2. Caso você já tenha o Java instalado, baixe apenas o arquivo BlueJ installer de acordo com o seu sistema operacional
  3. Execute o arquivo e siga os passos de instalação

Instalando o Eclipse

  1. Acesse a página de download do Eclipse EE clicando aqui
  2. No menu a direita, em Download links, selecione a versão de acordo com o seu sistema operacional
  3. Na próxima tela clique na setinha verde para realizar o download
  4. Após o download apenas extraia o arquivo em um local da sua preferência
  5. Pronto. Abra o eclipse para testar se deu certo.

Instalando o SourceTree

  1. Acesse a página de download do SourceTree clicando aqui
  2. Faça o download do aplicativo e execute o instalador
  3. Será necessário criar uma conta no GitHub para utilizar o programa durante as aulas

Instalando o OracleXE

  1. Acesse a página de download do Oracle XE clicando aqui
  2. Faça o download e execute o instalador
  3. Next -> Next -> [...] -> Finish
